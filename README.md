# Azure Retail Lakehouse Project

## Overview

This project demonstrates a production-style Azure Data Engineering solution using Azure Data Factory, Azure Databricks, PySpark, Delta Lake, Unity Catalog, Azure SQL Database, and Power BI.

The solution follows the Medallion Architecture (Bronze, Silver, Gold) to build scalable and reliable data pipelines.

---

## Tech Stack

- Azure Data Factory
- Azure Databricks
- PySpark
- Delta Lake
- Unity Catalog
- Azure Data Lake Storage Gen2
- Azure SQL Database
- Power BI
- Git & GitHub
