# Project Overview

## Business Scenario

ABC Retail is a multinational retail company receiving data from multiple systems every day.

The company wants to build a cloud-based modern data platform on Microsoft Azure.

This project demonstrates how to ingest, process, transform, validate, and serve analytics-ready data using Azure Data Factory, Azure Databricks, PySpark, Delta Lake, and Power BI.

## Objectives

- Build scalable ETL pipelines
- Implement Medallion Architecture
- Process incremental data
- Ensure data quality
- Build analytical datasets
- Create dashboards
